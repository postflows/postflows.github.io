What it does

TextPlus Manager gives you a panel to browse, select and edit every Text+ clip on your timeline without hunting through the timeline manually. Click any item in the list to jump directly to that clip.

Batch operations let you update font, size, or color across multiple selected clips at once ‚Äî ideal for large projects where branding changes late in the edit.

Installation
1
Download the script
Use the Download button above to get textplus-manager.lua
2
Place in Scripts folder
macOS: ~/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Comp/
3
Restart DaVinci Resolve
The script appears under Workspace ‚Üí Scripts menu
Windows path
C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\Comp\
